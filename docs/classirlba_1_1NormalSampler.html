Detailed Description

Sample from a normal distribution.

This performs pseudo-random sampling from the normal distribution, using the 64-bit Mersenne Twister from the C++ <random> standard library as the random number engine. However, the former may not be consistently implemented across different platforms; users may prefer to use the Boost normal_distribution implementation to ensure reproducibility across compilers.
