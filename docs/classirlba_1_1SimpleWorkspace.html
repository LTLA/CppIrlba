Detailed Description

template<class EigenVector_, class Simple_>
class irlba::SimpleWorkspace< EigenVector_, Simple_ >

Workspace class for multiplying a SimpleMatrix.

Template Parameters
EigenVector_A floating-point Eigen::Vector to be used as input/output of the multiplication.
Simple_The underlying matrix referenced by the SimpleMatrix.

Typically constructed by SimpleMatrix::new_workspace().
