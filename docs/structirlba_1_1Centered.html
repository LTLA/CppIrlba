Detailed Description

template<class Matrix>
struct irlba::Centered< Matrix >

Wrapper for a centered matrix.

Template Parameters
MatrixAn Eigen matrix class - or alternatively, a wrapper class around such a class.

This modification involves centering all columns, i.e., subtracting the mean of each column from the values of that column. Naively doing such an operation would involve loss of sparsity, which we avoid by deferring the subtraction into the subspace defined by rhs.
