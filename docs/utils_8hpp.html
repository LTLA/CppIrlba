Orthogonalize a vector against a set of orthonormal column vectors.

Parameters
matA matrix where the left-most ncols columns are orthonormal vectors.
vecThe vector of interest, of length equal to the number of rows in mat.
tmpA vector of length equal to mat.cols(), used to store intermediate matrix products.
ncolsNumber of left-most columns of mat to use.
Returns
vec is modified to contain vec - mat0 * t(mat0) * vec, where mat0 is defined as the first ncols columns of mat. This ensures that it is orthogonal to each column of mat0.
